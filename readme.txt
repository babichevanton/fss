Простейшее web-приложение, выводящее на страницу список авиарейсов
Для развертывания необходим сервер GlassFish
Для установки следовать инструкциям на https://glassfish.java.net/download.html
Приложение находится в папке System/war. Структурная часть лежит в папке WEB-INF

Структура WEB-INF:
1) classes/
Здесь лежат скомпилированные java-классы сущностей БД, DAO-классы и контроллеры страниц, а также конфигурационные 
файлы связывания для Hibernate.
2) lib/
Здесь находятся все библиотеки, необходимые для работы проекта.
3) pages/
Здесь лежат jsp-страницы, отображающие результат работы приложения в браузере.
4) web.xml
Конфигурационный файл Spring Framework
5) dispatcher-servlet.xml
Конфигурационный файл, управляющий настройкой Hibernate и настраивающий работу сервера

Для развертывания приложения необходимо выполнить из папки System команду
ant deploywar
После этого переместить полученный war-архив в папку glassfish4/glassfish/domains/domain1/autodeploy и дождаться ответа
сервера. Это будет или "fss.war_deployed", что означает успех, или "fss.war_deployFailed". В последнем случае нужно 
смотреть лог (../logs/server.log) и искать ошибку (с конца).

В случае успеха в браузере в адресной строке набрать
localhost:8080/fss/flights
и смотреть на ответ сервера. Если отрисовалась страница, то все хорошо. Иначе смотреть ответ сервера.
